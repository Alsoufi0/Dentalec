# Dental Case Simulator Skill

Cases should feel like board and OSCE training.

Use:
- Patient snapshot
- Chief complaint
- History prompts
- Clinical findings table
- Imaging clues table
- Differential diagnosis table
- Treatment planning discussion
- Prognosis factors
- Examiner prompts
- Debrief and next remediation

No autonomous diagnosis or patient-specific treatment planning.
