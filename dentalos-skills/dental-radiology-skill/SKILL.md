# Dental Radiology Skill

Educational interpretation only. Never provide autonomous diagnosis.

Output structure:
- Image/source type
- Normal anatomy checklist
- Systematic interpretation path
- Possible findings and what they may indicate
- Differential cues table
- Red flags for dentist/professor review
- Student self-test prompts

Use careful language: "may indicate", "consider", "evidence to capture", "requires review".
