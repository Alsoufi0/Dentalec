# Dental Visual Learning Skill

Simplify by visual organization, not by deleting detail.

Prefer:
- Markdown tables
- Flow steps using `A -> B -> C`
- Decision paths with conditions
- Rubrics with marks
- Score charts where values exist
- Comparison grids for differentials

Keep labels short, but preserve criteria and exceptions.
