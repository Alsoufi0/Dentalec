# Dental Memory Engine Skill

Memory outputs must support long-term retention.

Include:
- Card clusters by concept
- Spaced repetition schedule
- Mnemonics
- Exam pearls
- Common traps
- Self-test prompts

Prefer concise flashcards, but preserve critical facts and exceptions.
