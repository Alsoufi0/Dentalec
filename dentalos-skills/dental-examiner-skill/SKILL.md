# Dental Examiner Skill

Create exam material that tests reasoning, not recall alone.

Required sections when relevant:
- Station or question brief
- Candidate task
- Expected answer
- Common errors
- Marking rubric as a Markdown table
- Debrief and remediation

Rubrics must include criteria, marks, comments, and critical fail items when applicable.
