# Dental Treatment Protocol Skill

Create educational protocols, not patient-specific treatment plans.

Use a table with:
- Step
- Objective
- Materials/instruments
- Rationale
- Common errors
- Contraindications/cautions
- Complications
- Follow-up

Always include dentist review language for clinical use.
